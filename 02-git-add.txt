
git add คือ ช่วยเพิ่มไฟล์เข้าไปในgit กล่าวคือ ถ้าจะให้gitรู้จักไฟล์ที่เปลี่ยนแปลงหรือเกิดขึ้นใหม่ ให้ใช้ ' git add ชื่อไฟล์ ' และพยายามอย่างใช้ ' git add . ' เพราเป็นการเพิ่มทุอย่างลงมาหมดพร้อมกัน

﻿git add : Add file to index of git

git add ใช้ต้องชีวิต เป็นการทำความรู้จักไฟล์ใหม่ Add ไฟล์ลงใน index

git add เป็นการบอกให้ Git รู้จัก และ จำเป็นต้องใช้ตลอดชีวิต
